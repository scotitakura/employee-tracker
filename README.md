# Employee Tracker

## Purpose
Using Mysql to help you manage a team, position, or employee. Create, delete, or view your team easily though various command line prompts.

## Built With
* JS
* Node
* inquirer
* Mysql2

## Usage
To run program: npm start or node index.js
